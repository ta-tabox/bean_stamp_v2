## 移行資材について

本リポジトリには、旧リポジトリから抽出した移行資材を `docs/migration-resources` に取り込み済みです。  
以降の実装では旧リポジトリのソースコードを直接参照せず、以下を一次情報として開発を進めてください。

- `docs/NEXTJS_REPLACE_SPEC.md`
- `docs/migration-resources/**`
- `docs/issues/*.md`

## ドキュメント
